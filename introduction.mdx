## Overview

Paperguide provides a centralized space to organize and manage your project documentation. You create, edit, and collaborate on docs within customizable spaces. Start by setting up your first space to streamline your workflow.

## Key Features

You access a rich text editor with markdown support. Embed images, links, and code snippets easily. Use spaces to separate projects, like one for API docs and another for user guides.

<Steps>
  <Step title="Create Space" icon="plus">

    Navigate to the dashboard and click `Create New Space`.

    ```javascript
    // Example API call to create space
    const response = await fetch('/api/spaces', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ name: 'My Project' })
    });
    ```

  </Step>
  <Step title="Add Document" icon="file-text">

    Inside your space, select `New Document`.

    ```bash
    curl -X POST https://api.paperguide.com/documents \
      -H "Authorization: Bearer YOUR_TOKEN" \
      -d '{"title": "Getting Started", "content": "# Hello World"}'
    ```

  </Step>
  <Step title="Share Access" icon="share-2">

    Go to settings and invite collaborators.

  </Step>
</Steps>
